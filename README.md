# d3_graph_viewer
requires a .json formatted to generate graph. 

Sample format:
{
"nodes":[
{"name":"node1","group":1},
{"name":"node2","group":2},
{"name":"node3","group":1}
],
"links":[
{"source":1,"target":2,"value":1},
{"source":1,"target":3,"value":1},
{"source":2,"target":1,"value":1},
{"source":3,"target":2,"value":1},
]

}
